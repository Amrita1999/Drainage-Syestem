OBJECTIVE

We have designed a project for the Smart India Hackathon 2019 Hardware Edition whic is named as
Automatic Drainage Channel.In this project our focus is to determine the level of overflow of drains
and minimize the blockage condition by a simple method.To overcome the drainage problems in our society,
we have come up with an idea of reducing this problem and getting India one step closer to SMART INDIA

IDEATION

In this Project we have taken up the problem of Drainage and come up with a better idea.

Firstly we have set up nodes at 100 metres distance which consist of wires if different
length in descending order.These wires measure the level of water and if the level is more
the co-ordinates are shown in here.We have compared the adjacent nodes data to check if there
is any blockage condition.If yes,the details are shown in the map and the authority is informed about
the location having drainage overflow issue.

Secondly we have taken into consideration that the flowing materials in the drainage also cause overflowing.
So to minimize this we have set up filters at every 4 km distance which operates automatically via the servo motor
